Installation & Setup
Clone the repo:
git clone https://github.com/ranbir19/Mental_Health_Buddy.git

Install dependencies:
pip install -r requirements.txt

Get an Azure OpenAI API key and endpoint.

Update the Python script with your Azure API key and endpoint.

Usage
Run the script in VSCode or your Python environment:
python mentalbot.py
This will launch the Gradio chat interface where you can interact with Mental_Health_Buddy.
